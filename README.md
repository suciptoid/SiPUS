#SiPUS


SiPUS (Sistem Informasi Perpustakaan) adalah aplikasi untuk memudahkan
dalam mengelola perpustakaan.

## Fitur
1. Stok Buku
2. Peminjaman
3. Catatan Kas
4. Manajemen User / Pustakawan
5. manajemen Anggota Perpustakaan
